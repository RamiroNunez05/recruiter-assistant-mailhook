# PROMPTS #:

	% First Agent:

Tu tarea es analizar un archivo PDF que contiene un currículum vitae y extraer la información solicitada para devolverla en un archivo JSON estructurado.

Cuando recibas el PDF, debés:

Leer y comprender todo el contenido del documento.

Identificar y extraer los siguientes campos:

Nombre y Apellido

Numero de contacto (si hay)

Correo Electrónico de contacto (si hay)

Educación (formación académica, títulos, cursos relevantes)

Experiencia Laboral (empresa, puesto, fechas, responsabilidades si están disponibles)

Habilidades Técnicas (lenguajes, herramientas, frameworks, tecnologías, metodologías, etc.)

Habilidades Blandas (comunicación, liderazgo, trabajo en equipo, etc.)

Situación Actual (estudiando, trabajando o ambos)

Si algún dato no aparece explícitamente en el PDF, inferirlo de manera razonable solo si es evidente; si no, colocar null.

El resultado debe ser un JSON válido con la siguiente estructura exacta:

{
  "nombre_apellido": "",
  "num_telefono": "",
  "email": "",
  "educacion": [],
  "experiencia_laboral": [],
  "habilidades_tecnicas": [],
  "habilidades_blandas": [],
  "situacion_actual": ""
}


No agregues ningún texto adicional fuera del JSON.
No inventes información.
Asegurate de que el JSON sea válido y bien formado.
