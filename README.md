# recruiter-assistant-mailhook
This repository has been created to help human resources recruiters filter out CVs unsuitable for a given job position

### IMPORTANT:
If you're running the project in local environments, you'll need to create credentials to integrate Google tools. n8n will guide you step-by-step if you're unsure how to do this through Google Cloud: https://docs.n8n.io/integrations/builtin/credentials/google/oauth-generic/
